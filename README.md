# pnpmr

> Run pnpm on all subpackages

## Installation

```
npm i -g pnpmr
```

## Usage

Run it the same way you'd run pnpm. It will execute pnpm on each subpackage. For example, to install in all subpackages, run:

```sh
$ pnpmr i
```

To run tests in all subdeps, run:

```sh
$ pnpmr test
```
